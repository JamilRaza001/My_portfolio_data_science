SpaceX Launch Analysis & Prediction Dashboard

This project is an end-to-end machine learning solution that predicts the success of SpaceX launches, integrated into an interactive Streamlit dashboard. It fetches real-time data from the SpaceX API including launch details, rocket types and launchpad information and enhances it with simulated weather data (temperature, wind speed, and conditions) tied to launch dates and locations. The train_model.py script processes this data, encoding categorical variables like rocket and launchpad names with LabelEncoder, and trains a Random Forest Classifier on features such as weather conditions and temporal factors (year, month, day). The trained model and encoders are saved for deployment. The Streamlit app then loads these artifacts to offer a user-friendly interface with data filtering (by year and site), a Folium map visualizing launch site success rates, and analytical tabs showing rocket performance, weather impacts, and feature correlations. Users can also input custom parameters to predict launch success probabilities. This project impacts space enthusiasts and analysts by providing actionable insights into launch success factors, demonstrating robust data processing and machine learning pipelines, and offering an engaging tool for exploration all with a modular, maintainable codebase that could be extended with real weather data or advanced models.
